## 🧰 How to Run Any Project

1. **Navigate to the project folder:**

    ```bash
    cd [project-folder]
    ```

2. **Build and run the Docker container:**

    ```bash
    docker-compose up --build
    ```

3. **Visit in your browser:**

    ```
    http://localhost:8000
    ```

4. **Stop the container:**

    ```bash
    docker-compose down
    ```
